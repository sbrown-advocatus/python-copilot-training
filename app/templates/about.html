About CSV Parser & Visualizer

CSV Parser & Visualizer is a Flask-based web application designed to make it easy to work with CSV data.

Features

  • CSV file upload with support for various delimiters
  • Automatic parsing and data type detection
  • Interactive data tables with pagination
  • Column selection for customized views
  • Data filtering and sorting
  • Chart visualization for numerical data
  • Data statistics for numerical columns
  • Export functionality for processed data

How to Use

  1. Upload your CSV file using the Upload page
  2. View your data in the table view
  3. Apply filters and sort columns as needed
  4. Select specific columns to display
  5. Switch to chart view for visualizing numerical data
  6. View statistics for your data
  7. Download the processed data for further use
